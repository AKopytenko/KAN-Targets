# KAN-Targets

Задачник с сохранением задач в локальной памяти браузера и возможностью импорта/экспорта для переноса между устройствами.

## Установка
```
npm i
```

### Запуск на dev-сервере
```
npm run serve
```

### Сборка production
```
npm run build
```
